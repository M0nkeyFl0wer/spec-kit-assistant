## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Google Cloud account (optional, for cloud features)

### Installation

```bash
# Clone or create new project
npx spec-kit-assistant init my-project
cd my-project

# Install dependencies
npm install

# Start interactive consultation
npm start
```

### First Time Setup

```bash
# Interactive project setup with Spec's guidance
spec-assistant init --interactive

# Deploy agent swarm (3 agents)
spec-assistant swarm --deploy --scale 3

# Setup cloud integration with free tier optimization
spec-assistant cloud --setup --optimize

# Configure smart oversight
spec-assistant oversight --mode strategic
```

## 🔧 Configuration

### Environment Variables

```bash
# Optional: Enable voice synthesis
ELEVENLABS_API_KEY=your_api_key_here
SPEC_VOICE_ID=your_custom_voice_id

# Google Cloud Platform
GOOGLE_APPLICATION_CREDENTIALS=path/to/service-account.json
GCP_PROJECT_ID=your-project-id

# Agent Swarm
SWARM_MAX_AGENTS=10
SWARM_AUTO_SCALE=true

# Oversight
OVERSIGHT_MODE=strategic  # trust-verify | strategic | full-control
AUTO_APPROVAL_THRESHOLD=0.8
```

### Project Configuration

```json
{
  "spec-assistant": {
    "character": {
      "voiceEnabled": false,
      "personality": "friendly-golden-retriever"
    },
    "oversight": {
      "mode": "strategic",
      "criticalDecisionPoints": [
        "architecture-changes",
        "security-configurations",
        "database-schema-changes"
      ]
    },
    "cloud": {
      "provider": "gcp",
      "region": "us-central1",
      "optimizeForFreeTier": true
    },
    "agents": {
      "autoScale": true,
      "maxAgents": 10,
      "defaultScale": 3
    }
  }
}
```

## 🐛 Troubleshooting

### Common Issues

**Q: Spec isn't responding to my questions**
```bash
# Check if the character system is properly initialized
spec-assistant character --generate

# Verify Node.js version
node --version  # Should be 18+
```

**Q: Agent swarm deployment fails**
```bash
# Check Docker is running (if using containers)
docker --version

# Verify network connectivity
spec-assistant swarm --monitor
```

**Q: GCP integration not working**
```bash
# Verify authentication
gcloud auth list

# Check project access
gcloud projects describe YOUR_PROJECT_ID

# Test connectivity
gcloud compute instances list
```

**Q: Costs are higher than expected**
```bash
# Review current usage
spec-assistant cloud --costs

# Check budget alerts
spec-assistant cloud --optimize

# Review resource allocation
spec-assistant swarm --monitor
```
